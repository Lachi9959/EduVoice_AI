## Overview

EduVoice AI is an AI-powered educational platform that enables learners to generate lectures, practice interviews, get instant Q&A support, and study in multiple languages — all through natural voice interactions. It acts as a personal AI tutor available 24/7.

---

## Features

### 🎓 AI-Powered Lecture Generation
Generate comprehensive, context-aware lectures on any topic. The AI understands context and produces engaging educational content tailored to the learner's needs.

### 🎤 Voice Interview Practice
Simulate real interview scenarios with an AI agent that listens, responds, and provides instant feedback with improvement suggestions. Full interview history and detailed reports are available.

### 💬 Smart Q&A Assistant
Ask questions on any subject and receive clear, intelligent explanations in real time — like having a knowledgeable tutor always on hand.

### 🌍 Multi-Language Support
Learn and practice in multiple languages with native-level pronunciation guidance and grammar correction.

### 📊 Admin Dashboard
A comprehensive dashboard for user management, analytics, and platform oversight.

### 🔑 API Key Integration
Connect your own AI provider keys — supports **OpenAI**, **Google Gemini**, and **Anthropic Claude**.

### 🔒 Secure Platform
Enterprise-grade security with OAuth support for safe authentication and data handling.
